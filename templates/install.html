RTTA: Installation

Installation is via pip. To successfully install you will need a compiler and python source code on your machine. If you are using home brew you are probally okay. If you are running ubuntu and you've never compiled something before you'll need to take some steps first. 

  # // Steps for ubuntu users
  # sudo apt install build-essential python-dev

Once that's out of the way you'll be able to install via pip:

  # pip install rtta

That's all. Its installed
